## 安装
前提条件需在本地安装 api server 参考上面的api地址在本地启用。项目中默认监听的是4000端口，也可在我项目中fetch.js中进行修改配置
```shell
$ git clone git@github.com:liubaiye/react-music.git
$ npm install
```

## 运行

```shell
$ npm start
```

服务器启动默认端口为 3000,若不想使用 3000 端口,可使用以下命令: Mac/Linux

```shell
$ PORT=4000 node app.js
```
